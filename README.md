# Blogger

: 직접 사용하고 운영할 수 있는 블로그 

### 기능
1. 회원가입
2. 댓글 작성/수정/삭제
3. 게시글 검색
4. 사용자 권한 설정 : 스태프 권한이 있는 사람은 블로그 글을 쓸수 있도록
5. 소셜 로그인 기능
6. 인터넷 상에서 접속 가능
7. HTTPS인증

### 학습 목표
* 1장 - BootStrap
* 2장 - 장고 기본구조 MTV(Model, Template, View)
* 3장 - 테스트 주도 개발법 TDD 도입하여 게시글 수정, 댓글 기능 구현
     - 소셜 로그인 기능
* 4장 - 웹사이트를 인터넷에 공개 : AWS서버, 도메인 구입, Docker로 웹서비스 설정, HTTPS인증
